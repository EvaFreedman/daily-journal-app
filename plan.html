Daily Journal App

Architecture plan — learning project

Stack

LayerTechnologyWhy
Framework Next.js (App Router) Frontend + backend in one project; great for learning
Styling Tailwind CSS Fast to write, readable, widely used
Storage localStorage No database setup needed; simplest path to a working app
AI Claude API Powers the mood tagger feature
Env vars .env.local Next.js convention; API key lives here, server-side only

How .env Works (key idea)

  • .env.local stores ANTHROPIC_API_KEY=sk-...
  • This file is never sent to the browser — Next.js keeps it server-side only
  • The mood tagger calls /api/mood (a Next.js API route)
  • That route reads the key from process.env.ANTHROPIC_API_KEY and calls Claude
  • The browser only receives the mood result — the key stays hidden

Features

Core

Write entry
Textarea + save button, stored in localStorage with timestamp
Browse by date
List of entries sorted newest-first, grouped by date
Search by keyword
Live filter as you type

AI Feature

Mood tagger
After saving an entry, Claude returns a mood label + emoji — e.g. "Reflective 🤔", "Happy 😊", "Anxious 😬"

Data Shape (localStorage)

type JournalEntry = {
  id:    string // crypto.randomUUID()
  date:  string // "2026-04-01T10:30:00Z"
  text:  string // the entry content
  mood?: string // "Reflective 🤔" (optional)
}

Build Order

1
Run npx create-next-app@latest
Scaffolds the whole project for you
2
Build EntryForm + storage.ts
Write and save entries to localStorage
3
Build EntryList
Display entries, browse by date
4
Add search filter to EntryList
Live keyword filtering
5
Create /api/mood/route.ts + .env.local
Secure server-side call to Claude API
6
Wire up MoodBadge
Show mood tag after saving an entry

How to Know It Works

  • Write an entry → it still appears after a page refresh
  • Type in the search box → list filters in real time
  • Save an entry → mood badge appears (e.g. "Happy 😊")
  • Open browser DevTools → Network tab → ANTHROPIC_API_KEY is NOT visible
  • Check .env.local → it's listed in .gitignore (Next.js does this automatically)
